# How to Convert to PDF

Since automated PDF conversion tools aren't available, here are 3 simple methods to convert the user guide to PDF:

## Method 1: Using Your Browser (Recommended - Easiest)

1. **Open the HTML file**:
   - Navigate to: `c:\Users\danie\OneDrive\Documents\app  builds\operon-backup-2025-12-27_21-27-35\operon_user_guide.html`
   - Double-click to open in your default browser (Chrome, Edge, Firefox, etc.)

2. **Print to PDF**:
   - Press `Ctrl + P` (or go to File → Print)
   - In the print dialog, select **"Save as PDF"** or **"Microsoft Print to PDF"** as the destination
   - Click **"Save"** or **"Print"**
   - Save as `operon_user_guide.pdf` in the same folder

3. **Done!** You now have a professional PDF with proper formatting.

## Method 2: Using Microsoft Word (If you have Office)

1. Open Microsoft Word
2. Go to File → Open
3. Select `operon_user_guide.html`
4. Go to File → Save As
5. Choose **"PDF"** as the file type
6. Save as `operon_user_guide.pdf`

## Method 3: Online Converter

1. Visit a free HTML to PDF converter like:
   - https://www.html2pdf.com/
   - https://cloudconvert.com/html-to-pdf
2. Upload `operon_user_guide.html`
3. Download the converted PDF
